# All User Stories

This document provides a summary of all user stories generated for the project, grouped by their respective epics.

## Epic 1: Foundation & Core Task Management
- **1.1.Project_Scaffolding:** Set up the initial monorepo structure with a Spring Boot backend and React/Vite frontend.
- **1.2.User_Registration:** Allow new users to register for an account.
- **1.3.User_Login:** Enable registered users to log in and receive a JWT.
- **1.4.Basic_Task_Creation:** Allow authenticated users to create a new task with a title and description.
- **1.5.Kanban_Board_View_&_Interaction:** Display tasks on a Kanban board and allow users to change task status via drag-and-drop.
- **1.6.Task_Editing_and_Details:** Enable users to edit the details of a task.

## Epic 2: Task Collaboration & Gamification
- **2.1.Task_Commenting:** Allow users to add comments to tasks for collaboration.
- **2.2.File_Attachments_to_Tasks:** Enable users to attach files to tasks.
- **2.3.Foundational_Badge_System:** Implement a system for awarding and displaying badges based on user actions.

## Epic 3: Basic Project Management & Notifications
- **3.1.Project_Creation_and_Management:** Allow users to create and manage projects, including adding team members.
- **3.2.Linking_Tasks_to_Projects:** Enable the association of tasks with specific projects.
- **3.3.Basic_External_Notifications:** Allow project managers to configure and send notifications to external services like Slack or Teams.

## Known Gaps / Open Questions
- The exact criteria for all badges in the gamification system need to be defined by the Product Owner.
- The specific format of external notifications needs to be finalized.
